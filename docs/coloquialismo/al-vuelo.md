# Al vuelo

**Definición:**  
Realizar una tarea o tomar una decisión de manera rápida y/o improvisada, posiblemente sin planificación previa.

**Categoría:**  
Coloquialismo  

  


**Términos relacionados:**  
Bomberazo, Mods

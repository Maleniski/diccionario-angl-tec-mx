# Triggers

**Definición:**  
Conjunto de instrucciones SQL que se ejecutan automáticamente en respuesta a ciertos eventos en una tabla o vista de una base de datos.

**Categoría:**  
Bases de Datos  

  


**Términos relacionados:**  
SP, Primary, Foreign keys

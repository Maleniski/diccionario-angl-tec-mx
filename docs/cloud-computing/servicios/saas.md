# SaaS

**Definición:**  
Modelo de distribución de software donde las aplicaciones se alojan en la nube y se acceden a través de internet.

**Categoría:**  
Cloud Computing  

**Subcategoría:**  
Servicios

**Términos relacionados:**  
PaaS, IaaS

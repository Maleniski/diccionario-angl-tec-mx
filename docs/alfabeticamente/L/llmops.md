# LLMOps

Práctica de gestionar el ciclo de vida de modelos de lenguaje grandes (LLM) en producción, similar a MLOps.

**Categoría:**  
DevOps  

**Subcategoría:**  
Data Science

**Términos relacionados:**  
MLOps, DevOps

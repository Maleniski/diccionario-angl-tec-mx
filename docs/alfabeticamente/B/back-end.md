# Back-end

**Definición:**  
Parte de una aplicación que gestiona la lógica de negocio, bases de datos y las interacciones con la parte frontal.

**Categoría:**  
Desarrollo de Software  

**Subcategoría:**  
Arquitectura de Software

**Términos relacionados:**

# Base de datos

**Definición:**  
Conjunto organizado de datos almacenados electrónicamente en un sistema informático, accesibles para consultas y análisis.

**Categoría:**  
Bases de Datos  

  


**Términos relacionados:**  
Esquema de Base de datos, ETL

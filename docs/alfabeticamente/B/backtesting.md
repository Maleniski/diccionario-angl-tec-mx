# Backtesting

**Definición:**  
Proceso de probar una estrategia de inversión utilizando datos históricos para evaluar su viabilidad.

**Categoría:**  
Data Science  

**Subcategoría:**  
Finanzas

**Términos relacionados:**  
Forecasting, Clustering

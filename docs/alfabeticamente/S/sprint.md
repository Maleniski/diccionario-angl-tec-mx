# Sprint

**Definición:**  
Periodo de trabajo definido, generalmente de 1 a 4 semanas, durante el cual se desarrollan y entregan incrementos de un producto.

**Categoría:**  
Metodologías Ágiles  

  


**Términos relacionados:**  
Spike, Backlog

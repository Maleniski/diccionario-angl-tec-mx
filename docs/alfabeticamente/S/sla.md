*Término:* SLA  
*Definición:* Acuerdo entre un proveedor de servicios y un cliente que especifica los niveles de servicio esperados.

*Taxonomía:* Acuerdo de Nivel de Servicio  
*Términos relacionados:* Endpoint, API

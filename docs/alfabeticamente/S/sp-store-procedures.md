# SP (Store Procedures)

**Definición:**  
Conjunto de instrucciones SQL precompiladas que se almacenan y ejecutan en una base de datos para realizar operaciones específicas.

**Categoría:**  
Bases de Datos  

  


**Términos relacionados:**  
Triggers, Primary, Foreign keys

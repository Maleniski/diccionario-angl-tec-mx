# SAS Key

**Definición:**  
Clave de acceso compartido utilizada en Azure para conceder acceso limitado a recursos de almacenamiento.

**Categoría:**  
Cloud Computing  

**Subcategoría:**  
Seguridad

**Términos relacionados:**  
VM, VPN

# Sistemas Federados

**Definición:**  
Arquitectura de bases de datos en la que varios sistemas autónomos colaboran y comparten datos sin centralización.

**Categoría:**  
Bases de Datos  

**Subcategoría:**  
Data Science

**Términos relacionados:**  
Data Lake, Diccionario de datos

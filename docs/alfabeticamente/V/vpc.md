# VPC

**Definición:**  
Red virtual privada que permite la creación de redes lógicas aisladas dentro de un entorno de nube.

**Categoría:**  
Cloud Computing  

**Subcategoría:**  
Redes

**Términos relacionados:**  
VPN, SAS Key

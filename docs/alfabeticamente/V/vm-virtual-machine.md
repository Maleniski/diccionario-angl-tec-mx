# VM (Virtual Machine)

**Definición:**  
Instancia de un sistema operativo que se ejecuta sobre un hipervisor, simulando un hardware físico completo.

**Categoría:**  
Cloud Computing  

**Subcategoría:**  
Virtualización

**Términos relacionados:**  
Hiperescalador, VPC

# VPN

**Definición:**  
Tecnología que crea una conexión segura y cifrada sobre una red menos segura, como internet.

**Categoría:**  
Redes  

**Subcategoría:**  
Seguridad

**Términos relacionados:**  
VPC, SAS Key

# Front-end

**Definición:**  
Parte de una aplicación con la que interactúan los usuarios, generalmente a través de una interfaz gráfica.

**Categoría:**  
Desarrollo de Software  

**Subcategoría:**  
Arquitectura de Software

**Términos relacionados:**  
Back-end, Full-stack

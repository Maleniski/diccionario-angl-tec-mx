# Caché

**Definición:**  
Almacenamiento temporal de datos para acelerar su acceso futuro, evitando la necesidad de recalcular o recuperar información desde la fuente original.

**Categoría:**  
Optimización  

**Subcategoría:**  
Arquitectura de Software

**Términos relacionados:**  
Overhead, Pipeline

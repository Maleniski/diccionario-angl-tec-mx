# Cluster

**Definición:**  
Conjunto de servidores o nodos que trabajan juntos para proporcionar alta disponibilidad y escalabilidad.

**Categoría:**  
DevOps  

**Subcategoría:**  
Orquestación

**Términos relacionados:**  
Pod, Nodo

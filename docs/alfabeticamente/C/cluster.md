*Término:* Cluster  
*Definición:* Conjunto de servidores o nodos que trabajan juntos para proporcionar alta disponibilidad y escalabilidad.

*Taxonomía:* DevOps / Orquestación  
*Términos relacionados:* Pod, Nodo

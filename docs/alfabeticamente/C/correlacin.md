# Correlación

**Definición:**  
Medida estadística que indica la relación entre dos variables, utilizada para entender patrones en los datos.

**Categoría:**  
Data Science  

**Subcategoría:**  
Machine Learning

**Términos relacionados:**  
Causalidad, Clustering

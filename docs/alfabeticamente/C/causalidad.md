# Causalidad

**Definición:**  
Relación entre causa y efecto donde una variable influye directamente en otra, diferenciándose de la simple correlación.

**Categoría:**  
Data Science  

**Subcategoría:**  
Machine Learning

**Términos relacionados:**  
Correlación, Clustering

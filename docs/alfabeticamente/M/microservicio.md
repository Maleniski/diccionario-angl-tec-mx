# Microservicio

**Definición:**  
Arquitectura donde una aplicación se descompone en pequeños servicios independientes que interactúan entre sí.

**Categoría:**  
Arquitectura de Software  

  


**Términos relacionados:**  
Enterprise, Contenedor

# Metadatos

**Definición:**  
Información que describe otros datos, proporcionando contexto o detalles adicionales sobre su origen, estructura, o características.

**Categoría:**  
Bases de Datos  

**Subcategoría:**  
Data Science

**Términos relacionados:**  
Diccionario de datos, Dataset

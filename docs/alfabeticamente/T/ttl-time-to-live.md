# TTL (time to live)

**Definición:**  
Tiempo de vida de un paquete o dato antes de que sea descartado o eliminado.

**Categoría:**  
Desarrollo de Software  

**Subcategoría:**  
Redes

**Términos relacionados:**  
VPN, VPC

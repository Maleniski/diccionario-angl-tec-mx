# ETL

**Definición:**  
Proceso de extracción, transformación y carga de datos desde diversas fuentes a un destino final, como un data warehouse.

**Categoría:**  
Bases de Datos  

**Subcategoría:**  
Data Science

**Términos relacionados:**  
Clustering, Data Lake

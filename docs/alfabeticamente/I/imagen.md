# Imagen

**Definición:**  
Copia de un sistema operativo o aplicación, empaquetada para ser desplegada en contenedores u otros entornos.

**Categoría:**  
DevOps  

**Subcategoría:**  
Orquestación

**Términos relacionados:**  
Contenedor, Dockerizar

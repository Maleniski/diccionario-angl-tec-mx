# IaaS

**Definición:**  
Infrastructure as a Service; modelo de servicio que ofrece recursos de computación virtualizados a través de internet.

**Categoría:**  
Cloud Computing  

**Subcategoría:**  
Servicios

**Términos relacionados:**  
PaaS, SaaS

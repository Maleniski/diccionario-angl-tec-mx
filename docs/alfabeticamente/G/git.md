# Git

**Definición:**  
Sistema de control de versiones distribuido utilizado para seguir los cambios en el código fuente durante el desarrollo de software.

**Categoría:**  
Desarrollo de Software  

**Subcategoría:**  
Control de Versiones

**Términos relacionados:**  
Pipeline, Framework

# Gobierno de datos

**Definición:**  
Conjunto de prácticas y políticas para asegurar la gestión adecuada, calidad y seguridad de los datos en una organización.

**Categoría:**  
Data Science  

**Subcategoría:**  
Gobierno de Datos

**Términos relacionados:**  
Data stewards, Diccionario de datos

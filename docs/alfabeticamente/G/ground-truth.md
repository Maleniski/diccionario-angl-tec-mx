# Ground Truth

**Definición:**  
Datos de referencia utilizados para entrenar y evaluar modelos de machine learning, considerados como la verdad factual.

**Categoría:**  
Data Science  

**Subcategoría:**  
Machine Learning

**Términos relacionados:**  
Overfit, Underfit

# Homologación

**Definición:**  
Proceso de alineación de datos o procesos para cumplir con un estándar común o requisitos específicos.

**Categoría:**  
Data Science  

**Subcategoría:**  
Machine Learning

**Términos relacionados:**  
Normalización, Estandarización

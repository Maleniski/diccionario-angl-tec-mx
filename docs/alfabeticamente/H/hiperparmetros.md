# Hiperparámetros

**Definición:**  
Parámetros que se configuran antes del entrenamiento de un modelo de machine learning y que controlan su comportamiento y precisión.

**Categoría:**  
Data Science  

**Subcategoría:**  
Machine Learning

**Términos relacionados:**  
Ensamble, Imputación

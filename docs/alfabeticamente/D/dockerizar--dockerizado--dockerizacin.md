# Dockerizar / dockerizado / dockerización

**Definición:**  
Proceso de encapsular una aplicación en un contenedor Docker para facilitar su despliegue y ejecución.

**Categoría:**  
DevOps  

**Subcategoría:**  
Contenedores

**Términos relacionados:**  
Contenedor, Pod

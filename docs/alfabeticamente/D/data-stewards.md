# Data stewards

**Definición:**  
Profesionales responsables de la gestión, calidad y gobernanza de los datos en una organización.

**Categoría:**  
Gobierno de Datos  

**Subcategoría:**  
Data Science

**Términos relacionados:**  
Gobierno de datos, Diccionario de datos

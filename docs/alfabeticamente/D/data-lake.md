# Data Lake

**Definición:**  
Almacén de datos centralizado que permite almacenar datos en su formato original, tanto estructurados como no estructurados, para análisis y procesamiento posterior.

**Categoría:**  
Bases de Datos 
**Subcategoría:**  
Almacenamiento

**Términos relacionados:**

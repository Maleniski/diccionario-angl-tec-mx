# Diccionario de datos

**Definición:**  
Repositorio centralizado que contiene descripciones y definiciones de los elementos de datos de una base de datos.

**Categoría:**  
Bases de Datos  

**Subcategoría:**  
Data Science

**Términos relacionados:**  
Metadatos, Esquema de Base de datos

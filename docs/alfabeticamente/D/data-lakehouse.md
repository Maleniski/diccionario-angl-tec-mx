# Data Lakehouse

**Definición:**  
Arquitectura que combina las ventajas de un Data Lake y un Data Warehouse, permitiendo análisis avanzados y gestión de datos estructurados y no estructurados.

**Categoría:**  
Bases de Datos  

**Subcategoría:**  
Almacenamiento

**Términos relacionados:**  
Data Lake, ETL

# Precisión

**Definición:**  
Proporción de verdaderos positivos sobre el total de predicciones positivas, utilizada como métrica de evaluación en modelos de clasificación.

**Categoría:**  
Data Science  

**Subcategoría:**  
Machine Learning

**Términos relacionados:**  
F-Score, Matriz de Confusión

# PaaS

**Definición:**  
Plataforma como servicio que proporciona un entorno en la nube para desarrollar, gestionar y ejecutar aplicaciones sin gestionar la infraestructura subyacente.

**Categoría:**  
Cloud Computing  

**Subcategoría:**  
Servicios

**Términos relacionados:**  
SaaS, IaaS

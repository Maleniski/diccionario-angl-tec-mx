# Modelo Multivariado

**Definición:**  
Modelo de análisis estadístico o machine learning que considera múltiples variables para predecir un resultado o entender relaciones complejas entre ellas.

**Categoría:**  
Data Science  

**Subcategoría:**  
Machine Learning

**Términos relacionados:**  
Causalidad, Correlación

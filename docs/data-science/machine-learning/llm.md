# LLM

**Definición:**  
Modelos de Lenguaje Grandes; modelos de machine learning entrenados en grandes cantidades de texto, utilizados para tareas como generación de lenguaje natural.

**Categoría:**  
Data Science  

**Subcategoría:**  
Machine Learning

**Términos relacionados:**

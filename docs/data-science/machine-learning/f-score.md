# F-Score

**Definición:**  
Métrica que combina la precisión y el recall en un modelo de clasificación para balancear su rendimiento general.

**Categoría:**  
Data Science  

**Subcategoría:**  
Machine Learning

**Términos relacionados:**  
Matriz de Confusión, Precisión

# CNN

**Definición:**  
Red Neuronal Convolucional; tipo de red neuronal diseñada para procesar datos con estructura de cuadrícula, como imágenes.

**Categoría:**  
Data Science  

**Subcategoría:**  
Machine Learning

**Términos relacionados:**

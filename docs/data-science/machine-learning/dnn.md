# DNN

**Definición:**  
Red Neuronal Profunda; tipo de red neuronal con múltiples capas ocultas, utilizada para modelar relaciones complejas en los datos.

**Categoría:**  
Data Science  

**Subcategoría:**  
Machine Learning

**Términos relacionados:**  
CNN, LLM

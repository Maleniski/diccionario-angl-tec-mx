# EDA

**Definición:**  
Exploratory Data Analysis; proceso de análisis preliminar de los datos para descubrir patrones, anomalías y probar hipótesis antes de aplicar modelos.

**Categoría:**  
Data Science  

**Subcategoría:**  
Análisis de Datos

*Término:* SAS Key  
*Definición:* Clave de acceso compartido utilizada en Azure para conceder acceso limitado a recursos de almacenamiento.

*Taxonomía:* Cloud Computing / Seguridad  
*Términos relacionados:* VM, VPN

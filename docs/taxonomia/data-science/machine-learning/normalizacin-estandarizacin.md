# Normalización, Estandarización

**Definición:**  
Procesos de ajuste de datos para que se ajusten a una escala o distribución estándar, facilitando su uso en modelos de machine learning.

**Categoría:**  
Data Science

**Subcategoría:**  
Machine Learning

**Términos relacionados:**

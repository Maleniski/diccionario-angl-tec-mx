# Contenedor

**Definición:**  
Unidad estándar de software que empaqueta el código y todas sus dependencias para que una aplicación se ejecute de manera uniforme en cualquier entorno.

**Categoría:**  
DevOps

**Subcategoría:**  
Orquestación

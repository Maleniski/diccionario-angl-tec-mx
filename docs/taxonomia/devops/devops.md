# DevOps

**Definición:**  
Conjunto de prácticas que unifican el desarrollo de software (Dev) y las operaciones de TI (Ops) para mejorar la eficiencia y calidad del despliegue de aplicaciones.

**Taxonomía:**  
DevOps

# MVP

**Definición:**  
Producto Mínimo Viable; versión simplificada de un producto con suficiente funcionalidad para atraer a los primeros usuarios y validar una idea.

**Categoría:**  
Gestión de Proyectos 
**Subcategoría:**  
Desarrollo de Software

# Wireframe

**Definición:**  
Esquema básico o plano de una interfaz de usuario, utilizado para definir la estructura y la disposición de los elementos en la pantalla.

**Taxonomía:**  
Diseño / Desarrollo de Software

**Términos relacionados:**
